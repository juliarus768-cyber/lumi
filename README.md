# Lumi

Lumi is a child-centered learning platform built to help every child discover how they learn best.

## Brand Foundation

- Company: Lumi
- Core AI: Nova™
- Technology: Learning DNA™
- Philosophy: Confidence First™
- Tagline: "Helping Every Child Discover How They Learn Best."

## Documentation

The Lumi knowledge base lives in `docs/`. It is organized by discipline so product, education, AI, design, engineering, content, QA, roadmap, marketing, legal, and archived materials can evolve without losing context.

## Documentation Index

- `docs/00_MASTER_BIBLE/` - Source of truth for Lumi's mission, principles, terminology, and cross-functional decisions.
- `docs/01_PRODUCT/` - Product strategy, feature definitions, user journeys, and requirements.
- `docs/02_EDUCATION/` - Learning science, pedagogy, curriculum logic, and child development guidance.
- `docs/03_AI/` - Nova™ behavior, AI safety, personalization, prompts, and Learning DNA™ system notes.
- `docs/04_DESIGN/` - Brand, UX, UI, accessibility, and design system documentation.
- `docs/05_ENGINEERING/` - Architecture, development workflow, technical decisions, and implementation guidance.
- `docs/06_CONTENT/` - Voice, tone, lesson content, story systems, and content operations.
- `docs/07_QA/` - Test strategy, acceptance criteria, bug triage, and release validation.
- `docs/08_ROADMAP/` - Product planning, milestones, releases, and long-term vision.
- `docs/09_MARKETING/` - Messaging, positioning, campaigns, audiences, and launch materials.
- `docs/10_LEGAL/` - Privacy, compliance, policy, licensing, and child-safety documentation.
- `docs/99_ARCHIVE/` - Retired, superseded, or historical documents kept for reference.
