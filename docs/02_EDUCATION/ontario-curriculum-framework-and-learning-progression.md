## Math Adventure Ontario Curriculum Framework & Learning Progression

Version 1.0 Educational Blueprint

## Purpose

This framework defines how Math Adventure follows the Ontario mathematics curriculum while personalizing each child's learning journey. It specifies what is taught, when it is introduced, how mastery is measured, and how the adaptive engine supports long-term retention.

## Educational Principles

- Follow the Ontario curriculum as the source of truth.

- Review the completed grade before introducing new concepts.

- Promote mastery instead of memorization.

- Use adaptive review and spaced repetition.

- Teach through Tutor Mode rather than simply marking answers.

## Grade 4 → Grade 5

Learning progression for this pathway.

### Multiplication and division fluency

Learning Objective

Students demonstrate confidence and conceptual understanding of multiplication and division fluency through progressively more complex tasks.

Prerequisites

- Knowledge from previous grade

- Basic number sense

- Confidence with prerequisite operations

Common Misconceptions

- Rushing through calculations

- Confusing procedures with concepts

- Difficulty transferring knowledge to word problems

Tutor Strategy

- Visual explanation

- Guided questioning

- Worked example

- Independent retry

- Later review using spaced repetition

Mastery Criteria

- Consistently correct across multiple missions

- Can explain reasoning

- Applies skill in unfamiliar context

- Retains skill over time

Bridge to Next Concept

When mastery is achieved, gradually introduce the next related curriculum concept while continuing periodic review.

## Adaptive Learning Rules

- Every mission includes current-grade review and next-grade preparation.

- Weak skills appear more often but never dominate the mission.

- Mastered skills continue to appear occasionally to prevent forgetting.

- No child skips prerequisite concepts.

## Question Bank Requirements

- Every question tagged by grade, strand, skill, difficulty and misconception type.

- Each question includes one or more hints.

- Each question has a Tutor Mode explanation.

- Each question maps to Ontario curriculum expectations.

## Mission Composition

- Warm-Up: confidence building

- Skill Builder: targeted practice

- Adventure Round: mixed curriculum

- Boss Round: transfer and reasoning

## Future Expansion

This framework is intentionally curriculum-driven rather than question-driven. Future versions will extend to Grades 1–8 and additional subjects including Reading Adventure, Science Adventure, Financial Literacy Adventure and Homework Adventure while preserving the same adaptive progression model.

## Acceptance Criteria

- Every curriculum expectation maps to one or more measurable skills.

- Every skill has mastery criteria.

- Every mission supports review, progression and retention.

- Adaptive engine can determine what to teach next using this framework.
